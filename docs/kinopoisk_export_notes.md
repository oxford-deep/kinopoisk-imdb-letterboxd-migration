# Экспорт оценок из Кинопоиска

Этот шаг нужен, чтобы получить файл `ratings.csv`.

В репозитории есть готовый экспортёр:

```text
scripts/kinopoisk_votes_exporter.js
```

## Установка

Нужен Node.js 18+ и установленный Google Chrome или Microsoft Edge.

```bash
npm install
```

Экспортёр использует `puppeteer-core` и установленный браузер. Он не скачивает отдельный Chromium.

## Запуск

```bash
node scripts/kinopoisk_votes_exporter.js "https://www.kinopoisk.ru/user/<numeric_id>/votes/" ratings.csv
```

Лучше использовать числовой ID профиля:

```text
https://www.kinopoisk.ru/user/123456/votes/
```

## Как работает экспортёр

Скрипт:

```text
открывает браузер
переходит на страницу votes
даёт пройти логин / капчу вручную, если нужно
обходит оценки по бакетам 1/10 … 10/10
собирает фильмы и сериалы
сохраняет ratings.csv после каждой страницы
```

Обход по бакетам:

```text
/votes/list/vote/1/vs/vote/
/votes/list/vote/2/vs/vote/
...
/votes/list/vote/10/vs/vote/
```

Это важно: Кинопоиск может давать ссылки не только на следующие страницы фильмов, но и на персон/актёров. Поэтому экспортёр не ходит по произвольным “next”-ссылкам.

Если оценку не удалось распарсить из карточки фильма, `userVote` заполняется из текущего бакета.

CSV сохраняется в UTF-8 with BOM, чтобы Excel меньше ломал русские названия.

## Что должно получиться

Желательный набор колонок:

```text
dateTime,url,isSeries,name,originalName,year,duration,isWatched,userVote,rating,votes,ratingImdb,votesImdb
```

Минимально важные:

```text
name
originalName
year
userVote
```

## Если Кинопоиск покажет логин или капчу

```text
1. пройти проверку руками в открытом браузере
2. вернуться в терминал
3. нажать Enter
```

## Почему экспорт может дать меньше оценок, чем есть в профиле

Возможные причины:

```text
часть оценок относится к сериалам/ТВ-шоу
страницы имеют разную разметку
Кинопоиск поменял интерфейс
экспортёр не смог разобрать часть карточек
```

Для дальнейшей задачи это не всегда критично. Цель — не музейная точность архива, а перенос основного ядра оценок.

## Что проверить перед конвертацией

Перед запуском:

```bash
python scripts/kp_to_imdb.py ratings.csv
```

проверьте, что в CSV есть:

```text
userVote
year
originalName или name
```

Если Excel показывает кракозябры:

```text
Excel → Data → From Text/CSV → File Origin: 65001 UTF-8 → Delimiter: comma
```

## Главное

Рабочий путь:

```text
Puppeteer exporter
↓
числовой ID профиля
↓
обход оценок 1/10 … 10/10
↓
ручной проход логина / капчи при необходимости
↓
ratings.csv
```
