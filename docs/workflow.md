# Workflow: Кинопоиск → IMDb → Letterboxd

Это пошаговый маршрут переноса оценок из Кинопоиска в IMDb и Letterboxd.

Цель не просто “сохранить оценки”, а восстановить сценарий:

```text
много старых оценок
↓
профиль вкуса в Letterboxd
↓
люди с похожим вкусом
↓
Activity from friends как фильтр "смотреть / не смотреть"
```

---

## 0. Что понадобится

Перед началом нужны:

```text
Python 3.10+
браузер с DevTools
аккаунт IMDb
аккаунт Letterboxd
CSV с оценками Кинопоиска
```

Этот репозиторий начинается с момента, когда у вас уже есть CSV с оценками Кинопоиска.

Если CSV ещё нет, смотрите:

```text
docs/kinopoisk_export_notes.md
```

---

## 1. Подготовить CSV из Кинопоиска

Положите CSV с оценками в корень проекта и назовите его:

```text
ratings.csv
```

Желательный формат колонок:

```text
dateTime,url,isSeries,name,originalName,year,duration,isWatched,userVote,rating,votes,ratingImdb,votesImdb
```

Минимально важные колонки:

```text
name
originalName
year
userVote
```

Где:

```text
name          название на русском / локальное название
originalName  оригинальное название
year          год фильма
userVote      ваша оценка от 1 до 10
```

Пример входного файла:

```text
examples/kinopoisk_ratings_example.csv
```

---

## 2. Конвертировать оценки в IMDb format

Запустите:

```bash
python scripts/kp_to_imdb.py ratings.csv
```

Скрипт сделает три вещи:

```text
скачает IMDb dataset title.basics.tsv.gz, если его нет рядом
попробует найти IMDb ID по названию и году
создаст CSV для импорта оценок в IMDb
```

На выходе появятся файлы:

```text
imdb_import_ready.csv
matched_review.csv
unmatched.csv
```

Что это значит:

```text
imdb_import_ready.csv  файл для импорта в IMDb
matched_review.csv    список успешно найденных совпадений
unmatched.csv         список фильмов, которые не удалось найти
```

Формат `imdb_import_ready.csv`:

```csv
const,your rating
tt0133093,10
tt0111161,9
```

---

## 3. Проверить результат матчинга

Откройте:

```text
matched_review.csv
```

Проверьте несколько строк глазами:

```text
правильно ли найден фильм
совпадает ли год
не перепутан ли ремейк / сериал / короткометражка
```

Потом откройте:

```text
unmatched.csv
```

Не пугайтесь, если часть фильмов не нашлась.

Обычно не матчится:

```text
локальное кино
сериалы
ТВ-шоу
фильмы с другим английским названием
фильмы с ошибкой в годе
проекты, которых нет в IMDb
```

Для восстановления профиля вкуса не обязательно переносить 100% оценок. Важнее перенести основное ядро.

---

## 4. Необязательный шаг: добить часть unmatched через IMDb AKAs

IMDb хранит альтернативные названия фильмов в отдельном датасете `title.akas.tsv.gz`.

Это может помочь с фильмами, которые в IMDb называются иначе.

Запуск:

```bash
python scripts/kp_to_imdb_akas.py unmatched.csv
```

На выходе появятся:

```text
imdb_import_ready_akas.csv
matched_review_akas.csv
still_unmatched.csv
```

Важно: этот шаг тяжёлый.

`title.akas.tsv.gz` большой, поэтому скрипт может работать долго и сильно грузить ноутбук.

Практический совет:

```text
если первый скрипт уже перенёс большую часть оценок,
этот шаг можно пропустить
```

## 5. Импортировать оценки в IMDb

IMDb не даёт нормального официального CSV-импорта оценок.

Поэтому используется JS-скрипт для DevTools Console:

```text
scripts/imdb_bulk_rater_console.js
```

Порядок действий:

```text
1. Откройте https://www.imdb.com/
2. Залогиньтесь.
3. Откройте любую страницу IMDb.
4. Откройте DevTools.
5. Перейдите во вкладку Console.
6. Вставьте весь код из scripts/imdb_bulk_rater_console.js.
7. Нажмите Enter.
8. Выберите файл imdb_import_ready.csv.
9. Дождитесь завершения.
```

После завершения в консоли будет примерно:

```text
Done.
Successful: 1338
Failed: 0
```

Если вы делали дополнительный AKAs-матчинг, повторите импорт для файла:

```text
imdb_import_ready_akas.csv
```

---

## 6. Экспортировать оценки из IMDb

После импорта в IMDb:

```text
1. Откройте IMDb.
2. Перейдите в Your Ratings.
3. Нажмите Export.
4. Скачайте CSV с оценками.
```

Этот CSV нужен для импорта в Letterboxd.

---

## 7. Импортировать IMDb CSV в Letterboxd

В Letterboxd:

```text
1. Откройте Settings.
2. Перейдите в Import & Export.
3. Выберите импорт данных.
4. Загрузите IMDb CSV.
5. Дождитесь обработки.
```

Letterboxd переводит шкалу IMDb 1–10 в свою шкалу 0.5–5 звёзд:

```text
IMDb 10 → Letterboxd 5★
IMDb 9  → Letterboxd 4.5★
IMDb 8  → Letterboxd 4★
IMDb 7  → Letterboxd 3.5★
IMDb 6  → Letterboxd 3★
IMDb 5  → Letterboxd 2.5★
IMDb 4  → Letterboxd 2★
IMDb 3  → Letterboxd 1.5★
IMDb 2  → Letterboxd 1★
IMDb 1  → Letterboxd 0.5★
```

После этого профиль Letterboxd уже содержит вашу историю оценок.

## 8. Найти людей с похожим вкусом

Технический перенос оценок — это только половина задачи.

Дальше нужно найти людей, чьи оценки можно использовать как фильтр.

Рабочий способ:

```text
фильмы-маркеры
↓
рецензии с такой же оценкой
↓
профили авторов
↓
общие фильмы
↓
Follow
↓
Activity from friends
```

### 8.1. Выберите фильмы-маркеры

Хороший фильм-маркер — это не просто любимый фильм.

Хорошие маркеры:

```text
спорный фильм, который вы любите
фильм, который многие хвалят, а вы не любите
фильм, где ваша оценка сильно отличается от средней
фильм, который хорошо отражает ваш вкус
```

Плохие маркеры:

```text
фильмы, которые любят почти все
фильмы из универсальных топов
фильмы, где совпадение оценки почти ничего не говорит
```

### 8.2. Откройте фильм в Letterboxd

На странице фильма:

```text
1. Откройте Reviews.
2. Отфильтруйте рецензии по вашей оценке.
3. Откройте профиль автора рецензии.
```

Почему через рецензии:

```text
рецензии часто пишут активные пользователи
у таких пользователей обычно много оценок
профиль с 1000+ оценок полезнее, чем профиль с 50 фильмами
```

### 8.3. Проверьте профиль

В профиле пользователя:

```text
1. Оставьте только фильмы, которые смотрели вы оба.
2. Быстро пробегитесь по оценкам.
3. Посмотрите совпадения и сильные расхождения.
```

Человек подходит, если:

```text
совпадают важные любимые фильмы
совпадают важные нелюбимые фильмы
нет большого количества жёстких расхождений
у пользователя много оценок
профиль живой и активный
```

Если подходит — нажмите:

```text
Follow
```

## 9. Как пользоваться дальше

Когда вы подписались на несколько людей с похожим вкусом, не нужно каждый раз ходить по их профилям.

Открываете страницу фильма и смотрите блоки:

```text
Activity from friends
Reviews from people you follow
Watched by friends
Liked by friends
```

Это и есть практическая замена старой логики Кинопоиска:

```text
думаю смотреть фильм
↓
открываю Letterboxd
↓
смотрю оценки людей с похожим вкусом
↓
принимаю решение
```

Плюс Letterboxd начинает показывать активность друзей:

```text
кто что недавно посмотрел
кто что оценил
что можно добавить в watchlist
что можно сразу откинуть
```

## 10. Что делать с новыми оценками

Нормальной двусторонней синхронизации IMDb ↔ Letterboxd нет.

Есть три варианта:

```text
IMDb главный
Letterboxd главный
оценивать в двух местах
```

Практический вариант для этой задачи:

```text
Letterboxd = основной сервис для новых оценок
IMDb = архив / резервная база
```

Почему:

```text
социальная часть живёт в Letterboxd
Activity from friends живёт в Letterboxd
люди с похожим вкусом живут в Letterboxd
```

Если когда-нибудь понадобится перенести новые оценки обратно в IMDb:

```text
Letterboxd export
↓
конвертация
↓
IMDb GraphQL import
```
