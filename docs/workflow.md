# Полный workflow: Кинопоиск → IMDb → Letterboxd

## 0. Что должно получиться

Маршрут такой:

```text
Кинопоиск
↓
ratings.csv
↓
IMDb title id
↓
imdb_import_ready.csv
↓
IMDb ratings
↓
IMDb export
↓
Letterboxd import
↓
фильмы-маркеры
↓
люди с похожим вкусом
↓
Activity from friends
```

## 1. Экспорт оценок из Кинопоиска

Нужен CSV с колонками примерно такого вида:

```text
dateTime,url,isSeries,name,originalName,year,duration,isWatched,userVote,rating,votes,ratingImdb,votesImdb
```

Минимально важные колонки:

```text
name
originalName
year
userVote
```

Если у вас другой экспортёр, это нормально. Главное, чтобы были название, год и ваша оценка.

Подробности по экспорту и возможным проблемам:

```text
docs/kinopoisk_export_notes.md
```

## 2. Конвертация в IMDb CSV

Положите ваш `ratings.csv` в корень проекта.

Запустите:

```bash
python scripts/kp_to_imdb.py ratings.csv
```

Скрипт сам скачает IMDb dataset `title.basics.tsv.gz`, если файла нет рядом.

На выходе появятся:

```text
imdb_import_ready.csv    файл для импорта в IMDb
matched_review.csv      что удалось сматчить
unmatched.csv           что не удалось сматчить
```

## 3. Дополнительный матчинг через alternative titles

Этот шаг необязательный.

Он может помочь с фильмами, у которых в IMDb другое название.

Запуск:

```bash
python scripts/kp_to_imdb_akas.py unmatched.csv
```

Скрипт скачает `title.akas.tsv.gz`, если файла нет рядом.

На выходе:

```text
imdb_import_ready_akas.csv
matched_review_akas.csv
still_unmatched.csv
```

Важно: этот шаг тяжёлый. IMDb AKAs — большой файл. На слабом ноутбуке может быть долго и горячо.

## 4. Импорт оценок в IMDb

1. Откройте IMDb в браузере.
2. Залогиньтесь.
3. Откройте любую страницу IMDb.
4. Откройте DevTools.
5. Перейдите во вкладку `Console`.
6. Вставьте полный код из:

```text
scripts/imdb_bulk_rater_console.js
```

7. Нажмите Enter.
8. Появится выбор файла.
9. Выберите:

```text
imdb_import_ready.csv
```

10. Дождитесь завершения.

Если делали дополнительный AKAs-матчинг, повторите импорт для:

```text
imdb_import_ready_akas.csv
```

## 5. IMDb → Letterboxd

1. Откройте IMDb.
2. Перейдите в `Your Ratings`.
3. Нажмите `Export`.
4. Скачайте CSV.
5. Откройте Letterboxd.
6. Перейдите в `Settings`.
7. Откройте `Import & Export`.
8. Выберите импорт IMDb CSV.
9. Дождитесь обработки.

## 6. Как искать людей с похожим вкусом

Технический перенос оценок — это только половина задачи.

Цель — вернуть сценарий:

```text
открыл фильм
↓
посмотрел, что поставили люди с похожим вкусом
↓
понял, смотреть или нет
```

Рабочий способ:

1. Выберите 5–10 фильмов-маркеров.
2. Откройте страницу фильма в Letterboxd.
3. Откройте рецензии.
4. Отфильтруйте рецензии по вашей оценке.
5. Откройте профиль автора рецензии.
6. В профиле оставьте только общие фильмы.
7. Быстро пробегитесь по оценкам.
8. Если вкус похож — нажмите `Follow`.

Что такое хороший фильм-маркер:

```text
спорный фильм, который вы любите
фильм, который многие хвалят, а вы не любите
фильм, где ваша оценка сильно отличается от средней
фильм, который хорошо показывает ваш вкус
```

Плохие маркеры:

```text
фильмы, которые любят почти все
фильмы из топов, где совпадение ничего не значит
```

## 7. Как пользоваться дальше

Когда вы подписались на 5–10 похожих людей:

1. Открываете страницу фильма.
2. Смотрите блок `Activity from friends`.
3. Смотрите `Reviews from people you follow`.
4. По оценкам этих людей принимаете решение.

Это не идеальная копия старого Кинопоиска, но практическую задачу решает.
